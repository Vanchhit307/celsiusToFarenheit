# 🌡️ Celsius to Fahrenheit Converter (Java GUI)

## 📌 Description

This is a simple Java GUI application that converts temperature from **Celsius to Fahrenheit** using Java Swing.

---

## 🚀 Features

* User-friendly GUI interface
* Convert Celsius to Fahrenheit instantly
* Error handling for invalid input
* Lightweight and easy to run

---

## 🧮 Formula Used

Fahrenheit = (Celsius × 9/5) + 32

---

## 🛠️ Technologies Used

* Java
* Swing (GUI)

---

## ▶️ How to Run

1. Open the project in IntelliJ IDEA / Eclipse
2. Compile the program
3. Run the `celsiustofar` class
4. Enter temperature in Celsius
5. Click **Convert**
